开站记：向优秀博客学习的写作原则
发表于2025-11-20|开站记
阮一峰的《网络日志》、少数派的深度专题、硬核桃的产品评测，它们的共同点是：不追热点，只输出可复用的方法论。我希望借鉴这些优秀博客的节奏，让这个站点成为“技术+生活方式”的持续实践笔记。 我想做到的三件事 稳定节奏：参考阮一峰“每周五更新”的纪律，我会将技术长文定在每周二，灵感速记放在每周五。 结构化内容：学习少数派的专题写法，每篇文章都包含背景、拆解、落地三个部分，帮助读者快速迁移。 真实体验：借鉴 indie hackers 的“写给未来的自己”，只记录亲测有效的工具链、流程与心智模型。 写作模板（可复制开箱）1234567标题：一句话说明问题引子：为什么值得写 / 谁会受益正文： 1. 背景与判断 2. 拆解步骤（含工具、命令、截图） 3. 复盘与延伸阅读行动号召：下一篇要写什么 / 想听到什么反馈 Hexo 快速操作仍然留下一份最常用的 Hexo 命令清单，方便随时开写： 12345678# 新建文章hexo new "专题名"# 本地预览hexo server# 构建与部署hexo generate && hexo deplo...
